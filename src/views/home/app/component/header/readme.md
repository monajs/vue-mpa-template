## Header

### the handler of back option.
*** you can set track here. ***
```
meta: {
  backHandler: ()=>{...}
},
```

### change the title.
```
window.Header.setTitle('your title')
```

### show Header.
if you want to show the Header, you can update the meta config of route.js.
```
meta: {
  showHeader: true
},
```
